### **Experiment No. 11**

### **Title:**
Study of Dataset Creation and Data Analysis using Pandas

### **Aim:**
To create a dataset using Pandas and perform basic data analysis operations.



### **Theory:**
Pandas is an open-source Python library used for data manipulation and analysis.
It provides powerful data structures such as:
Series (1D) – Used to store one-dimensional data
DataFrame (2D) – Used to store data in tabular form (rows and columns)
It is widely used in data science for handling structured data efficiently.

###### Key Functions Used:
- DataFrame() → Used to create a dataset in tabular form from data like dictionary
- to_csv() → Saves the dataset into a CSV file so it can be stored and reused later
- read_csv() → Loads a dataset from a CSV file for analysis
- info() → Displays structure of dataset including data types, columns, and non-null values
- describe() → Provides statistical summary such as mean, count, minimum, and maximum
- head() → Shows first 5 rows of the dataset for quick viewing
- tail() → Shows last 5 rows of the dataset
- shape → Returns number of rows and columns in the dataset
- columns → Displays names of all columns
- sample() → Returns random rows from the dataset
- isnull() → Checks for missing values in the dataset
- duplicated() → Checks duplicate values present in the dataset
- nunique() → Counts number of unique values in each column

### **Conclusion:**
Pandas provides easy and efficient methods to create, store, and analyze datasets, making it a useful tool for data analysis.
